command

  • command(options?: commandOptions): Function
    • Defined in src/decorators/command.ts:19

    Use this method decorator to declare a class method a command

    class MyCommands {
        @command()
        ping(ctx: Context) {
             ctx.send("pong");
        }
}

    Using this decorator will automatically register your command function in commands

    Parameters

    • Optional options: commandOptions

      Here you can declare some metadata like the description or usage of your command

    Returns Function

reload

  • reload(path: string): Promise<{ message: string }>
    • Defined in src/reload.ts:8

    Under the hood this function will delete an already imported/required file from the cache and reimport it.
    It is ensured that registered commands that are part of a reloaded file file replace the old commands.
    If the file you are trying to reload is not found, a friendly promise rejection with an error message will be sent back.

    Parameters

    • path: string

      Path to be reloaded. Optimally, you should be using path.join

    Returns Promise<{ message: string }>
